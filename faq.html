FAQ

Risposte alle domande comuni

Il libro non si carica correttamente +

Verifica che il file EPUB sia valido. Puoi controllare IndexedDB in DevTools → Application → Storage → IndexedDB → EpubLibraryDB → books per vedere se il libro è stato salvato correttamente.

La posizione di lettura non viene salvata +

Usa il pulsante Save State (icona floppy) per salvare esplicitamente le preferenze. Verifica in DevTools → Application → Storage → Local Storage le chiavi epub-[bookTitle]-position.

L'export PDF fallisce +

Verifica la connessione internet (le librerie CDN sono necessarie). Controlla la console del browser per errori CORS. Prova la stampa diretta tramite il pulsante Print come alternativa.

La sidebar TOC non si apre +

Verifica che il TOC sia popolato aprendo la console del browser e controllando book.navigation.toc. Alcuni EPUB non hanno un indice dei contenuti definito.

Come funziona la traduzione? +

Click destro su qualsiasi testo → "Traduci in [lingua]". Il browser utilizza il servizio di traduzione integrato (Google Translate, Microsoft Translator, ecc.). La traduzione avviene direttamente nel browser senza invio a server terzi.

Quali formati di immagine supporta l'editor? +

L'editor supporta JPG, PNG, GIF, WebP e SVG. Le immagini vengono convertite in base64 inline per renderle completamente standalone nel documento esportato.

Workaround

Soluzioni ai problemi comuni

⚠️ CORS con server locale

Se apri il file direttamente dal filesystem, alcune funzionalità potrebbero non funzionare. Usa un server locale: python -m http.server 8000 nella cartella del progetto.

⚠️ IndexedDB piena

Se la libreria è piena, elimina alcuni libri. Controlla lo spazio usato in DevTools → Application → Storage. La quota tipica è 50-100MB.

⚠️ Traduzione non disponibile

Alcuni browser non supportano la traduzione integrata. Su Chrome/Edge funziona automaticamente. La traduzione richiede connessione internet.

⚠️ Immagini non visualizzate

Alcuni EPUB usano path relativi non standard. Prova ad estrarre il capitolo: le immagini dovrebbero apparire correttamente nel documento standalone.
