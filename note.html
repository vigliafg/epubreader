Problemi

Problemi Noti

OPEN EPUB con CSS complessi

Alcuni EPUB con fogli di stile molto complessi o non standard potrebbero non essere visualizzati correttamente. Stiamo lavorando per migliorare il parsing CSS.

WIP Immagini grandi

Immagini con dimensioni molto grandi (>4000px) potrebbero causare rallentamenti durante la navigazione. Soluzione in sviluppo.

CLOSED Navigazione su mobile

Problema risolto. La navigazione ora funziona correttamente su tutti i dispositivi mobili con il menu hamburger.

CLOSED Estrazione tabelle

Problema risolto. L'estrazione delle tabelle HTML ora funziona correttamente con struttura preservata.

Guide

Consigli d'uso

📖 Lettura efficace

  • Usa le frecce della tastiera o i pulsanti per navigare tra i capitoli
  • Regola la dimensione del font nelle impostazioni per il massimo comfort
  • Il tema chiaro/scuro si adatta alle condizioni di luce

📦 Sistema di raccolta

  • Seleziona il testo e clicca "Add Selection" per aggiungere chunk di testo
  • Clicca direttamente su immagini o tabelle per aggiungerle alla collezione
  • Usa il pannello ispezione per gestire (visualizzare/rimuovere) i chunks raccolti
  • Invia la raccolta all'editor HTML per ulteriori elaborazioni

✏️ Editor HTML

  • Pulisci il codice HTML estratto rimuovendo tag inutili
  • Esporta il contenuto come file HTML o testo semplice
  • Il codice generato è ottimizzato per la pubblicazione web

🔧 Problemi comuni

  • Se un EPUB non si carica, verifica che non sia protetto da DRM
  • Per problemi di visualizzazione, prova a cambiare tema o dimensione font
  • Per file molto grandi, attendi qualche secondo per il caricamento completo
