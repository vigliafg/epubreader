ePub Slimmer

Comprime le immagini

📤 Upload Semplice

Trascina il tuo file EPUB o selezionalo dal browser. Supporta file .epub standard.

🎚️ Qualità Regolabile

Slider per impostare la qualità di compressione (default 60%). Più bassa = file più piccolo.

⚡ Elaborazione Parallela

Elaborazione di 4 immagini in parallelo per velocità ottimale. Barra di progresso in tempo reale.

💾 Download Automatico

Download automatico del file compresso con suffisso "_slim.epub". Mostra la percentuale di riduzione.

📦 Multi ePub Slimmer

Elabora fino a 10 file contemporaneamente. Coda con stato individuale: pending, processing, done, error.

🔄 Progresso Globale

Barra di progresso che mostra quanti file sono stati elaborati sul totale. Statistiche finali per ogni file.

🗑️ Gestione Errori

Se un file fallisce, gli altri continuano l'elaborazione. Messaggi di errore chiari per ogni problema.

📚 Parsing TOC

Legge automaticamente la struttura del libro dall'OPF e mostra l'albero gerarchico dei capitoli.

☑️ Selezione Multipla

Seleziona uno o più capitoli da estrarre. Checkbox per scegliere esattamente cosa includere.

📄 Export Capitoli

Esporta i capitoli selezionati come file EPUB separati. Ogni capitolo diventa un libro standalone.

📦 JSZip Integration

Utilizza JSZip per manipolare il contenuto EPUB. Estrazione e ricostruzione del file completamente nel browser.

🔍 Validazione

Validazione del file EPUB caricato. Messaggi di errore per file corrotti o non validi.
